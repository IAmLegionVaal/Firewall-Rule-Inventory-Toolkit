# Firewall Rule Inventory Toolkit

A read-only PowerShell toolkit for Windows Firewall profile and rule inventory.

## Features

- Firewall profile status
- Enabled inbound and outbound rule inventory
- Rule action and direction summary
- CSV, JSON, and HTML reports

## Run

```powershell
powershell.exe -ExecutionPolicy Bypass -File .\Firewall_Rule_Inventory_Toolkit.ps1
```

## Safety

Read-only reporting only. No firewall settings are changed.
